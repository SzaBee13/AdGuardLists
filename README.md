# Adguard UnBlock and Block list library

This library is a collection of unblock and block lists for AdGuard. It is designed to be used with the AdGuard Home DNS server, but can also be used with other DNS servers that support custom block lists.

## Url

The lists are available at the following URLs:

- `https://adguardlists.szabee.me`
- `https://raw.githubusercontent.com/SzaBee13/AdGuardLists/main`

## Lists

The following lists are available:

- `AdGuard UnBlock List`: A list of domains that are unblocked by AdGuard. This list is designed to be used with the AdGuard Home DNS server to unblock certain domains that are blocked by default.
- `AdGuard Block List`: A list of domains that are blocked by AdGuard. This list is designed to be used with the AdGuard Home DNS server to block certain domains that are not blocked by default.

## Usage

To use the lists, simply add the URLs to your AdGuard Home DNS server or other DNS server that supports custom block lists. The lists will be automatically updated every 24 hours.

## Contributing

If you would like to contribute to the lists, please submit a pull request with your changes. Please make sure to follow the formatting of the existing lists and provide a clear description of your changes.
